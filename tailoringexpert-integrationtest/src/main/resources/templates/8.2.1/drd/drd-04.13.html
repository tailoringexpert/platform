DRD-04.13 - Derating Analysis/Parts Application Review (Stress Analysis)
Reference: ECSS-Q-ST-30-11C Rev.1 Space product assurance - Derating - EEE components
NPSL NASA Parts Selection List
Purpose: To demonstrate that components comply with the project derating requirements as detailed in ECSS-Q-ST-30-11C Rev.1 or any other approved derating procedure.
Content: This analysis has to be performed on each equipment containing EEE components. Inputs from initial thermal analysis shall be taken into account. On the other hand, the derating analysis shall be used to refine the thermal analysis in order to avoid too large margins. Non-compliances with the project derating requirements which cannot be settled are subject to a RFW.
Format: The format to be used for the Derating Analysis/Parts Application Review shall be similar to the annexed one and shall contain at least the same information.
