DRD-08.06 - Static Verification Procedure
Reference: N/A
Purpose: Description of usage of PolySpace Desktop and PolySpace Verifier for a static verification of the S/W source code.
Content: For a static verification of the S/W source code the tools PolySpace Desktop and PolySpace Verifier could be used. PolySpace Desktop is used for a static verification on unit level; PolySpace Verifier is used for a static verification on integration level.
The output of a static verification on integration level fulfills totally the required content of the Static Verification Reporting.
